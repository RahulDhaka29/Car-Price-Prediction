About This Project

This Car Price Prediction application is a machine learning project designed to estimate the selling price of used cars based on a variety of features. It serves as a practical demonstration of a complete data science workflow, from data analysis to model deployment.

Technology Stack

  • Backend: Python, Flask
  • Machine Learning: Scikit-learn, Pandas, NumPy
  • Frontend: HTML, Tailwind CSS
  • Core Model: A Random Forest Regressor model, chosen after comparing multiple algorithms for the best performance.

Project Workflow

  1. Data Exploration (EDA): Analyzed the dataset to understand trends and correlations, such as the relationship between car age and price.
  2. Feature Engineering: Created new, powerful features like `car_age` from existing data to improve model accuracy.
  3. Data Preprocessing: Converted categorical data (like brand and fuel type) into a numerical format using one-hot encoding.
  4. Model Training & Evaluation: Trained and compared multiple regression models (Linear Regression, Random Forest, XGBoost) and selected the best performer based on the R-squared metric.
  5. Web Deployment: Built a user-friendly web interface using Flask and Tailwind CSS to allow users to interact with the trained model in real-time.
